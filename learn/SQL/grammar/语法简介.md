# SQL 语句分为DQL（数据查询语言）、DML（数据操纵语言）、DDL（数据定义语言）几类。

# * DDL 建表语句

`CREATE TABLE score (
  student_name varchar(40),
  chinese_score int,
  math_score int,
  test_date date
);
`
显示数据库中有那些表 `\d`

查看score表的定义情况`\d score`

`postgres=# \d score
               Table "public.score"
    Column     |         Type          | Modifiers 
---------------+-----------------------+-----------
 student_name  | character varying(40) | 
 chinese_score | integer               | 
 math_score    | integer               | 
 test_date     | date                  | 
`
